# 5118 API 通用错误码

## 返回结构

```json
{
  "errcode": "0",
  "errmsg": "",
  "data": { ... }
}
```

`errcode` 为 `"0"` 表示成功。

## 通用错误码

| errcode | 含义 |
|---------|------|
| `0` | 成功 |
| `100101` | 调用次数不够，请充值 |
| `100102` | 每秒调用量超过限制 |
| `100103` | 每小时调用量超过限制 |
| `100104` | 每天调用量超过限制 |
| `100202` | 请求缺少 APIKEY |
| `100203` | 无效的 APIKEY |
| `100204` | API 不存在 |
| `100208` | 请求方式不支持（需使用 POST） |
| `100403` | 您输入的apikey不正确 |

## 系统级错误码

| errcode | 含义 |
|---------|------|
| `200103` | 不存在的任务 |
| `200104` | 数据获取中（异步接口任务处理中，需轮询重试） |
| `200107` | 服务器超时 |
| `200121` | 该域名暂不支持导出 |
| `200201` | 传进参数为空 |
| `200202` | 用户ID为空 |
| `200203` | 请输入要查询的关键词 |
| `200204` | 请提交要查询的网址 |
| `200301` | 请提交正确格式的网址 |
| `200401` | 提交关键词的数量已达上限，一次最多能提交50个关键词 |
| `200405` | 请输入要检测的关键词，多个用"\|"隔开 |
| `200500` | 内容长度不能超过5000个字符 |
| `2005050` | 您输入的内容超过指定长度，不含html标签限5000字符，含html标签限10000字符 |
| `2004050` | 某关键词长度超过了指定的字符串长度 |
| `2004050` | 如输入多个关键词时请用"\|"进行分隔 |

## 错误码格式说明

错误码由三部分组成：

```
    2       002       01
    │        │        │
    │        │        └── 具体错误代码
    │        └────────── 服务模块代码（即数据ID）
    └─────────────────── 服务器错误（1:为系统级别错误）
```

## 调用注意事项

- 所有接口仅支持 **POST** 请求
- 必须在 Header 中携带 `Authorization: 你的APIKEY`
- Content-Type 使用 `application/x-www-form-urlencoded; charset=utf-8`
- 返回值为 JSON 格式
- 接口有频率限制（每秒/每小时/每天），超限会返回对应错误码
