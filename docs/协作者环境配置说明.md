# 协作者环境配置说明

本项目不要把真实 API Key 提交到 Git。协作者拉取代码后，应复制 `.env.example` 为 `.env`，再通过私密渠道获取真实密钥。

```powershell
copy .env.example .env
```

## 必填密钥

### 1. Qwen / 通义千问

用途：产品画像、Prompt 生成、主内容分析、文章定位分析。

```env
DASHSCOPE_API_KEY=真实QwenKey
DASHSCOPE_BASE_HTTP_API_URL=
QWEN_SEARCH_MODEL=qwen-plus
QWEN_ANALYSIS_MODEL=qwen3.7-max
QWEN_WRITING_MODEL=qwen-plus
QWEN_TIMEOUT_SECONDS=120
```

### 2. 豆包 / 火山方舟

用途：多 AI 推荐排名、品牌诊断、竞品对比、声量估算。

```env
DOUBAO_API_KEY=真实豆包Key
DOUBAO_RESPONSES_API_URL=https://ark.cn-beijing.volces.com/api/v3/responses
DOUBAO_MODEL=doubao-seed-evolving
DOUBAO_TIMEOUT_SECONDS=90
```

### 3. 元宝 / 腾讯混元

用途：多 AI 推荐排名、品牌诊断、竞品对比、声量估算。

```env
YUANBAO_API_KEY=真实元宝或混元Key
YUANBAO_RESPONSES_API_URL=https://tokenhub.tencentmaas.com/v1/responses
YUANBAO_MODEL=hy3
YUANBAO_TIMEOUT_SECONDS=30
```

也兼容：

```env
HUNYUAN_API_KEY=真实混元Key
TENCENT_HUNYUAN_API_KEY=真实混元Key
```

### 4. DeepSeek

用途：多 AI 推荐排名、品牌诊断、竞品对比；部分文章分析可扩展使用。

```env
DEEPSEEK_API_KEY=真实DeepSeekKey
DEEPSEEK_CHAT_API_URL=https://api.deepseek.com/chat/completions
DEEPSEEK_MODEL=deepseek-chat
DEEPSEEK_TIMEOUT_SECONDS=30
```

### 5. 媒介库

用途：同步媒介资源、匹配报价、估算发布成本。

```env
MEIJIEKU_BASE_URL=https://api.meijieku.com
MEIJIEKU_MOBILE=真实手机号
MEIJIEKU_PASSWORD=真实密码
MEIJIEKU_TOKEN=
MEIJIEKU_RESOURCE_PAGE_SIZE=100
MEIJIEKU_RESOURCE_MAX_PAGES=20
MEIJIEKU_MOCK_MODE=false
```

## 可选密钥

```env
BAIDU_API_KEY=真实百度千帆Key
BAIDU_SEARCH_API_URL=https://qianfan.baidubce.com/v2/ai_search/web_search
BAIDU_SEARCH_EDITION=lite
BAIDU_SEARCH_SOURCE=baidu_search_v2
BAIDU_SEARCH_TOP_K=5
BAIDU_TIMEOUT_SECONDS=30
```

也兼容：

```env
BAIDU_QIANFAN_API_KEY=真实百度千帆Key
BAIDU_SEARCH_API_KEY=真实百度千帆Key
```

### SerpApi

旧流程保留字段，当前一键分析主流程一般不需要。

```env
SERPAPI_API_KEY=真实SerpApiKey
```

## 成本控制

正式跑客户报告前确认这些值，避免一次分析调用过多：

```env
GEO_AI_PROMPT_COUNT=5
GEO_AI_BRAND_DIAGNOSTIC_PROMPT_COUNT=3
GEO_AI_COMPARISON_PROMPT_COUNT=2
GEO_AI_RECOMMENDATIONS_PER_PROMPT=10
GEO_AI_VISIBILITY_BRAND_LIMIT=10
GEO_AI_SOURCE_LINK_LIMIT=80
GEO_AI_ARTICLE_FETCH_LIMIT=40
GEO_AI_ENABLE_PROMPT_DISCOVERY=true
```

日常开发测试建议：

```env
GEO_AI_PROMPT_COUNT=2
GEO_AI_BRAND_DIAGNOSTIC_PROMPT_COUNT=1
GEO_AI_COMPARISON_PROMPT_COUNT=1
GEO_AI_RECOMMENDATIONS_PER_PROMPT=5
GEO_AI_VISIBILITY_BRAND_LIMIT=5
GEO_AI_SOURCE_LINK_LIMIT=20
GEO_AI_ARTICLE_FETCH_LIMIT=10
GEO_AI_ENABLE_PROMPT_DISCOVERY=false
```

## 安全规则

1. `.env` 已在 `.gitignore` 中，不要提交。
2. 不要把真实 Key 写进 `.env.example`、代码、文档、测试文件。
3. 如果 Key 曾经被提交到 Git，需要立即去对应平台后台轮换/作废。
4. 推荐用 1Password、Bitwarden、飞书密钥、企业微信私密消息等方式发送 `.env` 内容。
5. 给协作者最好开子账号或限额 Key，避免主账号密钥泄漏后不可控。
